Factura de Traslado
Versión Enterprise Edition

Este modulo permite incorporar la Facturación por Traslado:

Configuración

Dentro de la Factura debemos Activar el Campo CFDI Traslado



Funcionalidad

Una Factura de Traslado contiene los siguientes puntos:

  • ★   Clave de Producto o Servicio 43212105
  • ★   Unidad de medida
  • ★   Descripcion de la Mercancia que estamos trasladando
  • ★   Valores en 0.0
  • ★   No requiere impuestos


Como Paso Final el Timbrado de una Factura de Traslado agrega un nuevo atributo:



Este campo se ve reflejado tambien dentro del Reporte.


Customizaciones Fiscales Odoo CE y EE
